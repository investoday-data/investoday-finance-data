## 沪深京最新实时日行情

接口路径：`stock-quote/realtime`
请求方式：`GET`
tool_id：`get_stock_quote_realtime`

接口说明：获取指定沪深京股票的最新实时行情数据，包含股票当前价格、涨跌幅、开盘价、最高价、最低价、成交量、成交金额等核心数据，同时提供行业代码和行业名称信息，便于大模型进行即时的市场监测和股票分析，单次请求需指定股票代码。时间格式为yyyy-MM-dd HH:mm:ss

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `002594` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `002594` |
| `stockName` | 股票名称 | `比亚迪` |
| `marketType` | 市场类型(沪-sh、深-sz、京-bj) | `sz` |
| `openPrice` | 开盘价 | `359.89` |
| `closePriceYDay` | 昨日收盘价 | `361.99` |
| `currentPrice` | 当前价格 | `353.86` |
| `changeRatio` | 涨跌幅 | `-0.02245918` |
| `highPrice` | 最高价 | `359.89` |
| `lowPrice` | 最低价 | `353.62` |
| `dataTime` | 数据时间 | `2025-06-12 13:40:21` |
| `sysTime` | 系统时间 | `2025-06-12 13:40:30` |
| `status` | 状态 | `0` |
| `dealStockAmount` | 成交股数 | `100071` |
| `dealMoney` | 成交金额 | `3563408640.0` |
| `limitUpTime` | 涨停时间 | — |
| `limitDownTime` | 跌停时间 | — |
| `limitUpPrice` | 涨停价 | `398.19` |
| `limitDownPrice` | 跌停价 | `325.79` |
| `circulationValue` | 流通市值 | `420782222502.59` |
| `totalValue` | 总市值 | `1075403847801.6344` |
| `changeRatio1W` | 一周涨跌幅 | `-0.01561285` |
| `changeRatioB10D` | 10日涨跌幅 | `-0.01880209` |
| `changeRatioB1M` | 1月涨跌幅 | — |
| `industryCode` | 行业代码 | `280500` |
| `industryName` | 行业名称 | `乘用车` |
| `industryLV1Code` | 一级行业代码 | `280000` |
| `industryLV1Name` | 一级行业名称 | `汽车` |
| `dealMoney1W` | 近一周成交额 | — |
| `dealStockAmount1W` | 近一周成交量 | — |
| `turnOverRate` | 换手率 | `1.8` |

**接口示例**

```bash
investoday-api stock-quote/realtime stockCode=002594
```

## 股票实时资金流向

接口路径：`stock-quote/capital-flow`
请求方式：`GET`
tool_id：`get_stock_realtime_fund_flow`

接口说明：通过股票代码获取指定股票的实时资金流向数据，包括超大单净流入、大单净流入、主力净流入等关键指标，以及股票名称和行情日期，用于实时监控和分析个股的资金动向。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `002594` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `002594` |
| `stockName` | 股票名称 | `比亚迪` |
| `date` | 行情日期 | — |
| `largeNetInflow` | 大单净流入 | — |
| `mainNetInflow` | 主力净流入（超大单+大单） | — |
| `superLargeNetInflow` | 超大单净流入 | — |

**接口示例**

```bash
investoday-api stock-quote/capital-flow stockCode=002594
```

# 沪深京数据 / 基础信息

---

#### 股票基本信息（GET）

接口：`get_stock_basic_info`　
请求方式: `GET`　
tool_id: `get_stock_basic_info`

> 📌 GET 版本，兼容历史调用，仅支持单标的查询。推荐使用 POST 版本进行批量查询。

使用股票代码查询沪深京市场A股、B股的详细基本信息，包括股票代码、交易所代码、所属板块名称、股票简称及全称、上市状态、上市日期、退市日期、股票类型、公司内部ID、总股本、无限售流通股、无限售流通A股、办公地址、主营业务、最新财报日期、公司成长阶段等核心数据，可用于股票档案构建、投研基础信息获取、上市公司背景分析等场景。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `000001` |

**接口示例**

```bash
node scripts/call_api.js stock/basic-info stockCode=000001
```

#### 证券特别提示（GET）

接口：`list_stock_special_notice`　
请求方式: `GET`　
tool_id: `list_stock_special_notices`

> 📌 GET 版本，兼容历史调用，仅支持单标的查询。推荐使用 POST 版本进行批量查询。

通过股票代码和日期范围查询沪深京股票的特别提示信息，包括股票名称、上市状态、特别处理状态、变动原因、相关日期等核心公告数据，用于监控股票重大事件和风险提示。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `000001` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
node scripts/call_api.js stock/special-notices stockCode=000001
```

#### 股票基本信息（POST）

接口：`get_stock_basic_info`　
请求方式: **`POST`**　
tool_id: `get_stock_basic_info`

> 📌 POST 版本，支持批量查询（可传多个标的）。

通过股票代码查询沪深京A股或B股的基本信息，包括股票名称、全称、所属交易所、上市板块、上市状态、上市日期、退市日期、股票类别、公司ID、总股本、流通股本、A股流通股本、公司办公地址、主营业务以及最新财务报告日期，适用于股票信息查询和基本面分析。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | — | string | 股票代码【与stockCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `stockCodes` | — | array | 股票代码列表【与stockCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '600519']` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: stockCode, stockCodes, pageNum, pageSize
node scripts/call_api.js stock/basic-info --method POST
```

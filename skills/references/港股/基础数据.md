# 港股 / 基础数据

---

#### 港股基本信息（GET）

接口：`get_hk_stock_basic_info`　
请求方式: `GET`　
tool_id: `get_hk_stock_basic_info`

> 📌 GET 版本，兼容历史调用，仅支持单标的查询。推荐使用 POST 版本进行批量查询。

通过股票代码查询香港股票的基本信息，包括股票代码、名称（中文全称、简称、英文简称）、上市状态、上市板块、交易货币、买卖单位、上市日期、摘牌日期、公司代码、交易市场编码以及是否为港股通标的等核心数据，便于大模型进行股票基本信息的查询和分析。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `00001` |

**接口示例**

```bash
node scripts/call_api.js hk-stock/basic-info stockCode=00001
```

#### 港股基本信息（POST）

接口：`get_hk_stock_basic_info`　
请求方式: **`POST`**　
tool_id: `get_hk_stock_basic_info`

> 📌 POST 版本，支持批量查询（可传多个标的）。

通过股票代码（可批量）查询香港股票的基本信息，包括股票代码、名称（中文全称、简称、英文简称）、上市状态、上市板块、交易货币、买卖单位、上市日期、摘牌日期、公司代码、交易市场编码以及是否为港股通标的等核心数据，便于大模型进行股票基本信息的查询和分析。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCodes` | — | array | 股票代码 | `['00001', '00004']` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: stockCodes, pageNum, pageSize
node scripts/call_api.js hk-stock/basic-info --method POST
```

#### 上市公司互动问答

接口：`list_stk_consultations`　
请求方式: `GET`　
tool_id: `list_stk_consultations`

根据股票代码和日期范围查询上市公司在互动平台上的问答记录，包括咨询类型、提问人、问题详情、回复内容及传闻核实情况，支持分页获取，便于分析公司沟通与市场传闻。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `002594` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
node scripts/call_api.js stock/consultations stockCode=002594
```

| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash

```

#### 基金公司的公告

接口：`list_fund_announcements`　
请求方式: **`POST`**　
tool_id: `list_fund_announcements`

支持通过基金代码（必填）、公告ID、日期范围（格式yyyy-MM-dd）、标题（支持模糊查询），结合页码、页长进行分页查询基金公司公告，可返回公告发布日期、公告标识、基金代码及名称、标题、风险类型、公告来源、公告内容类别代码等信息，适用于金融资讯展示、企业公告管理及数据分析等场景，实现精准高效的公告信息获取与处理。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2020-01-02` |
| `fundCodes` | ✅ | array | 基金代码 | `['000001', '000004']` |
| `title` | — | string | 标题（支持模糊查询） | `贵州茅台利润大涨` |
| `announcementID` | — | integer | 公告id | `9128035` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, title, announcementID, pageNum, pageSize
node scripts/call_api.js fund/announcements --method POST fundCodes=000001 fundCodes=000004
```
